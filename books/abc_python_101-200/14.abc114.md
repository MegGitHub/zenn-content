# A - 753
問題はこちら。
https://atcoder.jp/contests/abc114/tasks/abc114_a

コードは以下です！自分の提出は[こちら](https://atcoder.jp/contests/abc114/submissions/27010355)。

```python: A.py
X = int(input())

if X == 3 or X == 5 or X == 7:
    print('YES')
else:
    print('NO')
```


## コード解説
$X$が$3, 5, 7$の時は、YESと出力し、その他はNOと出力できるように、`if~else文`で記述します。
